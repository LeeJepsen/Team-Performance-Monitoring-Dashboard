## How to Run This Dashboard 
This Shiny app can be run locally on any machine with R installed.

### 1) Install prerequisites
- Install **R** (recommended version: 4.2+)
- Install **RStudio** (recommended)

### 2) Download this repository
** (GitHub):**
- Click the green **Code** button → **Download ZIP**
 	- Unzip the folder somewhere you can find it

**OR**

- Go to Data folder and download dataset
- Go to App folder and download CMJappEx.R
- Save both in known location

### 3) Open the project in RStudio
- In RStudio: **File → Open Project** (if you have an `.Rproj`)
  - OR 
- In RStudio: Open file → Go to folder location and click `CMJappEx.R`

### 4) Install required packages 

### 5) Import the dataset by updating the file path to match the location of Data (excel file), Line 30 in code.

### 6) Run code (Control + Enter) or “Run-App” at top
